Buttons

For buttons, apply a class of butt to the element. For a blue button, also add butt-blue. For red, add butt-red. You get the idea.

Inputs

Inputs & labels are styled automatically to fill the width of their container. If you don't like it, strip out the inputs section of the stylesheet.
For highlighted labels & inputs, add a class of in-highlight. For errors, add a class of in-error
